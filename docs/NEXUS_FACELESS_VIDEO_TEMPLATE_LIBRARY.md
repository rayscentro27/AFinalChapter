# Nexus Faceless Video Template Library

## Purpose
Reusable faceless templates for YouTube Shorts, Instagram Reels, TikTok, and educational YouTube videos using transcript-driven source material and free/low-cost tools.

## Template 1: Hook + 3 Tips + CTA
1. Best use case: actionable education.
2. Ideal platform: Shorts/Reels/TikTok.
3. Target length: 30-60s.
4. Structure: hook -> tip1 -> tip2 -> tip3 -> CTA.
5. Hook style: problem-first.
6. Scene outline: title card + numbered tip slides.
7. Caption style: short bullets.
8. CTA style: save/share/follow.
9. Free tools: CapCut free, Canva free.
10. Auto-select when: high actionability score + evergreen topic.

## Auto-Selection Rules (System)
- Use highest platform-fit + quality-score template.
- Block templates requiring unapproved claims.
- Enforce per-bucket weekly distribution caps.
- Prefer lower-production-effort template when score tie.

## Safety Rules
- No fabricated claims.
- No unverified medical/legal/financial guarantees.
- Trading domain remains education/research only.
- Client-sensitive data excluded by default.
