# Nexus Faceless Video Template Library

Status: reusable template catalog. Free-tool-first and transcript-driven.

## Template 1: Faceless Narrated Breakdown
- best use case: educational explainers
- ideal platform: YouTube Shorts, Reels, TikTok
- target length: 30-60s
- structure: hook -> 3 points -> CTA
- hook style: "Most people miss this..."
- scene outline: title card, point 1/2/3, CTA outro
- caption style: bold keywords + simple lines
- CTA style: save/share/comment keyword
- free tools: CapCut, Canva, Pexels
- auto-select when: high-confidence educational topic
